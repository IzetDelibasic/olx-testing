# OLX Testing

Automation tests for the OLX platform.

## About

This project was developed as part of the Formal Methods course at university.

**Team members:**

- [Mirnes Muratspahić](https://github.com/MirnesMuratspahic)

## Installation

```bash
git clone <repository-url>
cd olx-testing
npm install
```

## Running Tests

```bash
npm test
```
